Good instinct adding the “modern + clean + white” constraint—that’s exactly what makes lo-fi look intentional instead of messy.

I’ll describe **what each component should visually look like on screen** so you can recreate it fast in Figma without overthinking.

---

# 🧱 1. Foundations

### **Container / Card**

Looks like:

* A **soft white rectangle floating on a light gray page**
* Background: pure white (#FFFFFF)
* Page behind it: very light gray (#F7F7F7)
* Rounded corners (8–12px)
* Inside: content spaced evenly (text blocks, buttons)

Visual:

```
[        Title         ]
[  short description   ]
[   button      →      ]
```

👉 Feels like a “tile” or “card” sitting on the page

---

### **Section / Frame**

Looks like:

* Full-width invisible container
* Stacks multiple cards vertically
* Clean spacing between items (16–24px)

Visual:

```
[ Section Title ]

[ Card ]
[ Card ]
[ Card ]
```

👉 Think “page structure,” not visible UI

---

# 🔘 2. Inputs & Actions

### **Button**

Looks like:

* Rounded rectangle
* Solid dark gray or black fill
* White centered text

```
[   Continue   ]
```

Variants:

* Primary: filled dark
* Secondary: white with gray border
* Disabled: faded (low opacity)

---

### **Text Input**

Looks like:

* White box with light gray border
* Slight rounding
* Gray placeholder text inside

```
[   Enter your name...   ]
```

States:

* Focus: darker border
* Error: red border

---

### **Search Bar**

Looks like:

* Pill-shaped (fully rounded ends)
* Light gray fill
* Small search icon on left

```
[ 🔍  Search...             ]
```

---

### **Dropdown**

Looks like:

* Same as input
* Small arrow (chevron) on right

```
[  Select option     ⌄  ]
```

When open:

```
[ Option 1 ]
[ Option 2 ]
[ Option 3 ]
```

---

### **Checkbox**

Looks like:

```
[ ] Option
[x] Option
```

* Small square
* Checked = filled or checkmark

---

### **Radio Button**

Looks like:

```
( ) Option
(•) Option
```

---

### **Toggle Switch**

Looks like:

```
[  ○──── ]  (off)
[ ────○ ]  (on)
```

* Pill track
* Circle slides left/right

---

# 🧠 Final Design Rules (what makes it look “modern”)

Everything should feel:

* **White base**
* **Soft gray accents**
* **Consistent spacing (8px grid)**
* **Same border radius everywhere**
* **Minimal clutter**
* **No heavy shadows**

---

# ⚡ Mental shortcut

If you’re ever unsure:

* Everything = rectangles + text
* Rounded slightly
* Spaced evenly

That’s literally 90% of UI design.
